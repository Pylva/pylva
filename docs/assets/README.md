# README Assets

`pylva-flow.png` is a high-resolution static workflow diagram generated from
`generate-pylva-flow-image.py`. `pylva-core-features.png` is the matching
core-features illustration generated from `generate-pylva-core-features-image.py`.

From the repository root:

```bash
python3 docs/assets/generate-pylva-flow-image.py
python3 docs/assets/generate-pylva-core-features-image.py
```
